# AI-Enhanced Automated Timetable Generator

## Overview
A full-stack automated timetable generation system designed to create
conflict-free academic schedules for multiple departments.

## Tech Stack
Backend: Python, Flask  
Frontend: React  
Database: MySQL  
Cloud: Azure  

## Features
- Automated conflict detection and resolution
- Modular backend architecture
- RESTful APIs for timetable generation
- Scalable design for multiple departments

## Architecture
The backend follows a layered architecture separating routing,
business logic, and data access for maintainability and scalability.

## Results
Reduced manual timetable preparation effort by approximately 80%.

## Future Enhancements
- Authentication and role-based access
- Advanced optimization algorithms
- Cloud scaling
